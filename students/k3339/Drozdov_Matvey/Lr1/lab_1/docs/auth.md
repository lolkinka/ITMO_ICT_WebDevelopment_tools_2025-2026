# Авторизация и Безопасность

API использует механизм защиты на основе **JWT (JSON Web Token)**. Большинство эндпоинтов требуют передачи валидного токена в заголовке запроса.

## Как получить доступ

1. **Регистрация**: Создайте нового пользователя через эндпоинт `POST /register`. Пароль будет автоматически захэширован перед сохранением в базу.
2. **Логин**: Отправьте `username` и `password` на эндпоинт `POST /login`. В ответ сервер вернет `access_token`.
3. **Авторизация**: Вставьте полученный токен в заголовок следующих запросов:
   `Authorization: Bearer <ваш_access_token>`
