# Справочник API

Основные группы эндпоинтов приложения:

## Пользователи (/users)
- `POST /register` — Создание нового аккаунта.
- `POST /login` — Получение JWT токена.
- `GET /users/me` — Получение данных своего профиля (*Требует Auth*).
- `PATCH /users/me/password` — Смена пароля текущего пользователя (*Требует Auth*).

## Счета (/accounts)
- `GET /accounts` — Список всех счетов текущего пользователя.
- `POST /accounts` — Создание нового счета.
- `GET, PATCH, DELETE /accounts/{id}` — Управление конкретным счетом.

## Транзакции (/transactions)
- `GET /transactions` — Список транзакций по всем счетам.
- `POST /create_transaction` — Создание транзакции.

## Категории и Бюджеты
- `GET /categories` — Просмотр доступных категорий.
- `POST /budget` — Установка лимита трат на категорию.
