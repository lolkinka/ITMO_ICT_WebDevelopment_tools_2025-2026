# Развертывание и запуск системы

Весь оркестр сервисов полностью контейнеризирован и автоматизирован. Развертывание, настройка сети и управление зависимостями выполняются с помощью **Docker Compose**.

## Требования к окружению

Перед запуском убедитесь, что на вашей хост-машине установлены:
* **Docker Desktop** (включает в себя Docker Engine и Docker Compose)
* Свободные порты на хосте: `8000` (API), `5433` (PostgreSQL), `6379` (Redis)

---

## Пошаговый быстрый старт

### Шаг 1: Подготовка файлов
Убедитесь, что структура вашего проекта соответствует требованиям. Файлы зависимостей `requirements.txt` должны содержать библиотеки `celery` и `redis` как для основного API, так и для парсера.

### Шаг 2: Сборка и запуск
Откройте терминал в корневой папке проекта (`Lr3/`) и выполните команду для принудительной сборки образов и запуска контейнеров:

```bash
docker compose up --build
```

### Шаг 3: Проверка статуса контейнеров
Откройте новое окно терминала и проверьте, что все четыре контейнера успешно запустились и работают:
```bash
docker compose ps
```
